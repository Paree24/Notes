# Suggestions

* It is recommended to complete all the courses in Fundamentals, Mandatory Theory and Math Required to support your course
* Try to complete 3 learning paths
* It is recommended to learn at least 2 languages from the programming section
* It is recommended to do a major project in each learning path

Note: All the courses are selected such that it is possible to obtain a free certificate (as far as possible). If you want to dwell deeper, look for online materials and courses at Coursera
